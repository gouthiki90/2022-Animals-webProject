## 입양 후기 게시판

### 모델

1. POST 모델
   1. 입양후기 POST
   2. 자유게시판 POST
   3. 지역별 정보 공유 게시판 POST
2. USER 모델
3. Comment 모델
   1. 테이블을 하나로 통일할 것인가
      1. 나중에 모든 댓글이 필요할 때 가져오기 유용함
   2. 테이블을 게시글 섹션 대로 나눌 것인가
      1. 연관관계를 짜기 쉬움
      2. 댓글을 가져올 때 효율적이다.
